Help Book!

How to use this App efficiently?

Well,We suggest that you follow your routine and track it daily so as to measure your productivity status and enhance it thereafter.

How to start using?

Well first you'll have to add some tasks for you to work upon ,then you may start any of your task accordingly to your priority. (Note:While this app doesn't sort tasks according to priority but still we'll suggest to go from highest to lowest)

Functionalities Overview!

Start:

Pressing it will trigger the timer and a window to ask your task will prompt.

Stop:

Pressing it will stop the timer.

Reset:

Pressing it will prompt a window which will ask if you want to switch to some other task and/or this task's done or you're going to continue with the same task.

Resume:

This will resume the timer.

CompletedTask:

This will show you with details the tasks you've done lately or more previous(if present).

Task stat:

This will show you the graph of your tasks done.

Break-Time:

This is a window which we'll prompt within every 45 minutes to stop you from overloading.

How to use the task statistics or completed tasks window?

To view the data, You first select the month and click on getDates button and then the dates will be there on the choosable dates menu.Then you can choose to see data of any date that you want.
